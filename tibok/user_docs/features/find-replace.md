# Find and Replace

tibok includes a built-in find and replace feature powered by macOS's native find bar.

## Opening Find

### Find
- **Menu**: Find > Find...
- **Shortcut**: Cmd+F
- Opens the find bar at the top of the editor

### Find and Replace
- **Menu**: Find > Find and Replace...
- **Shortcut**: Cmd+Option+F
- Opens find bar with replace field

## Using Find

### Basic Search
1. Press Cmd+F to open find bar
2. Type your search term
3. Matches are highlighted in the editor
4. Use the navigation buttons or shortcuts to move between matches

### Navigation
| Action | Shortcut |
|--------|----------|
| Find Next | Cmd+G |
| Find Previous | Cmd+Shift+G |

### Options
The find bar provides options for:
- **Case sensitive**: Match exact capitalization
- **Whole words**: Only match complete words

## Using Replace

1. Press Cmd+Option+F to open find and replace
2. Enter search term in the first field
3. Enter replacement text in the second field
4. Use buttons to:
   - **Replace**: Replace current match and find next
   - **Replace All**: Replace all matches at once

### Replace Shortcuts
| Action | Shortcut |
|--------|----------|
| Replace | Cmd+= |
| Replace All | (Click button) |

## Tips

- **Incremental search**: Results update as you type
- **Escape to close**: Press Escape to dismiss the find bar
- **History**: Previous searches may be available in the find field dropdown
- **Regex support**: The native find bar supports regular expressions

## Keyboard-Only Workflow

1. Cmd+F - Open find
2. Type search term
3. Cmd+G - Jump to next match
4. Cmd+Shift+G - Jump to previous match
5. Escape - Close find bar
